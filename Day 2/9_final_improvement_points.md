## ⭐ Important Points

- `/` always returns a float.
- `//` floors towards **negative infinity**.
- `%` returns the remainder.
- `**` is evaluated **right to left**.
- `=` → Assignment
- `==` → Compare values
- `is` → Compare object identity (memory reference).
- Comparison operators always return `True` or `False`.

### Operator Precedence

PEMCNAO

P → Parentheses

E → Exponent

M → *, /, //, %

C → Comparison

N → not

A → and

O → or

## ❌ My Mistakes

- Forgot to use `+=`; wrote `a + 10` instead.
- Used `=` instead of `==` for comparison.
- Used `is` instead of `==` in one comparison question.
- Predicted `20 // 3` incorrectly.
- Used short variable names (`a`, `b`, `o`).

---

## ✅ Improvements

- Use descriptive variable names (`age`, `username`, `operator`).
- Use `+=` when asked for assignment operators.
- Remember:
  - `==` → value comparison
  - `is` → object identity
- Think before running the code.
- Explain **why**, not just the answer.
