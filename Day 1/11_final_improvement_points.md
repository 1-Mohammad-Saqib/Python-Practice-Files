# 🐍 Day 1 – Variables & Data Types

## 📚 Concepts

### Variables
- Store values in memory.
- Use meaningful variable names.

### Data Types
- int
- float
- str
- bool
- list
- tuple
- set
- dict

### Input & Output
- `input()`
- `print()`
- f-strings

### Type Casting
- `int()`
- `float()`
- `str()`
- `bool()`

### type()
- Used to check the data type of a variable.

---

## ⭐ Important Points

- `input()` always returns a **string**.
- Use `int()` or `float()` to convert user input.
- `=` is an **assignment** operator.
- Use **f-strings** for formatted output.
- Use `type()` to check a variable's data type.
- Follow **snake_case** for variable names.

---

## ❌ My Mistakes

- Used `User_name` instead of `user_name`.
- Used `Name`, `Age`, `City` instead of `name`, `age`, `city`.
- Didn't use f-strings in every program where I could have.

---

## ✅ Improvements

- Always use **snake_case**.
- Prefer meaningful variable names over `a`, `b`, `x`.
- Use f-strings whenever possible.
- Think about the data type before performing operations.
- Explain **why** the output is correct, not just the output.
